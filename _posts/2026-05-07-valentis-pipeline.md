앞선 글에서는 데이터 변환을 통해 입력-출력 구조를 구성하는 과정을 살펴봤다.

이번 글에서는 그 다음 단계인 자동 분석 기능을 다룬다.
조건과 목표를 정의한 뒤 Run All을 실행하면, 여러 분석이 순서대로 자동 수행되고 통합 보고서가 생성된다.

---

## 왜 한 번에 실행해야 하는가

민감도 분석, 메타모델, 신뢰성 해석, 파레토 최적화는 각각 독립적으로도 의미 있는 분석이다.
하지만 실제 설계 검토에서는 순서대로 이어지는 하나의 흐름에 가깝다.

먼저 민감도 분석으로 중요한 파라미터를 파악한다.
그 다음 메타모델로 설계 공간을 예측하고, 신뢰성 해석으로 목표 달성 가능성을 확인한다.
마지막으로 파레토 최적화를 통해 서로 충돌하는 목표 사이의 트레이드오프를 찾는다.

이 과정을 모듈별로 따로 실행하면 설정을 반복해야 하고, 결과를 서로 연결하는 작업도 수작업으로 처리해야 한다.

Valentis의 일괄 자동 분석은 이 전체 흐름을 한 번에 실행하도록 구성되어 있다.

---

## 조건과 목표 정의

데이터 변환에서 파일을 불러오면 자동 분석 설정 카드가 활성화된다.

실행 전에 정의해야 할 항목은 크게 두 가지다.

### 제약조건과 목표

파라미터를 선택하고 조건(< / > / =)과 값을 입력한 뒤 추가한다.

항목은 두 가지 유형으로 구분된다.

CON (제약조건) — 반드시 만족해야 하는 조건
예: 응력 < 200

OBJ (목표) — 최적화 방향과 가중치를 지정하는 조건
예: 변위 최소화, w=1

가중치 w가 0보다 크면 OBJ로, 값만 입력한 항목은 CON으로 분류된다.

OBJ로 설정한 항목은 파레토 최적화의 목적함수로 자동 연동된다.
따라서 파레토 분석 단계에서 동일한 목표를 다시 입력할 필요가 없다.

### 신뢰성 설정

신뢰성 해석에 사용할 불확실성 분포와 Monte Carlo 샘플 수를 선택한다.

지원되는 분포는 다음과 같다.

- Uniform
- Normal
- Lognormal

Monte Carlo 샘플 수는 기본값으로 10,000개가 사용된다.

신뢰성 해석에서 사용할 제약조건은 앞서 입력한 CON 항목에서 자동으로 가져온다.

---

## Run All — 7단계 순차 실행

설정이 완료되면 **Run All**을 실행한다.

이후 7단계 분석 흐름이 순서대로 진행되며, 각 단계의 상태가 실시간으로 표시된다.

- **Sensitivity** — 입력과 출력 간 Pearson/Spearman 상관계수 계산
- **Design Explorer** — 제약조건과 목표 기준으로 최적 케이스 탐색
- **Meta Model** — 메타모델 학습 및 교차검증
- **Sobol** — Sobol 민감도 지수 계산
- **Adaptive Sampling** — 불확실성이 높은 영역의 추가 샘플 포인트 추천
- **Reliability** — Monte Carlo 기반 목표 달성 확률 계산
- **Pareto** — 다목적 최적화 및 파레토 프론트 도출

각 단계가 완료될 때마다 체크 표시가 붙는다.
오류가 발생하면 해당 단계에서 실행이 멈추고 원인이 표시된다.

---

## 통합 보고서

모든 단계가 완료되면 **Open Report** 버튼이 나타난다.

버튼을 클릭하면 새 탭에서 통합 보고서가 열린다.

보고서에는 자동 분석의 핵심 결과가 하나의 문서로 정리된다.

- 민감도 분석 결과
- 디자인 탐색 결과 및 최적 케이스
- 메타모델 학습 결과와 예측 정확도
- 신뢰성 해석 결과
- 파레토 최적화 결과

보고서에는 파일명, 입력/출력 파라미터 수, 케이스 수, 생성 일시도 함께 기록된다.
우측 상단의 PDF 저장 버튼을 통해 바로 출력할 수 있다.

---

## 개별 탭에서 이어서 확인

자동 분석 실행 후에는 각 분석 탭에 결과가 자동으로 반영된다.

통합 보고서에서는 전체 흐름을 한 번에 확인할 수 있고,
특정 분석을 더 자세히 보고 싶다면 해당 탭에서 추가 설정과 시각화를 이어서 작업할 수 있다.

예를 들어 민감도 분석 결과를 더 자세히 보거나,
메타모델 설정을 조정하거나,
파레토 프론트에서 특정 설계안을 비교하는 식으로 후속 검토가 가능하다.

---

## 마무리

해석 데이터를 불러오고, 조건과 목표를 정의한 뒤, Run All을 실행한다.

그러면 7단계 분석이 순차적으로 수행되고, 핵심 결과는 하나의 통합 보고서로 정리된다.

분석마다 설정을 반복하거나, 결과를 수작업으로 취합하는 과정이 줄어든다.

Valentis 원클릭 자동 분석은 해석 데이터를 설계 의사결정에 바로 활용하기 위한 흐름을 자동화한다.
